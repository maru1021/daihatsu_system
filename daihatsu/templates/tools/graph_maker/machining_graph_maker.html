適合率（他ファイルの平均パターンとの一致度）

計算方法:
①Leave-One-Out平均パターンを算出:
評価対象ファイルを除く他ファイルの
各X軸位置での平均値を計算
②パターン類似性スコアの計算:
相関係数スコア = max(0, r × 100)
※ r は評価ファイルと平均パターンの相関係数
③値の近さスコアの計算:
修正RMSE = √(パターン誤差² + (値の乖離 × 20)²)
スコア = (1 - 修正RMSE / 最大修正RMSE) × 100
④各軸の適合率を算出:
適合率 = 類似性スコア × 0.3 + 近さスコア × 0.7
⑤総合適合率の算出:
適合率(計) = (X軸 + Y軸 + Z軸 + S軸) / 4
各指標の意味:
パターン類似性:
波形の形状がどれだけ似ているかを示す。相関係数が高いほど波形パターンが類似。
値の近さ:
実際の値がどれだけ近いかを示す。修正RMSEが小さいほど値が近い。
適合率(計):
全軸の平均値。総合的な平均パターンとの一致度を表す。
波形特徴量:
最大負荷、平均負荷、変動係数、ピーク位置などZ-TCMD負荷の特性値。
変数の説明:
xi: 評価対象ファイルの各X軸位置での値
yi: 他ファイルの各X軸位置での平均値
n: データポイント数
r: 相関係数（-1～1）
RMSE: Root Mean Square Error（二乗平均平方根誤差）
Σ: 全データポイントの合計
適合率の解釈:
80%以上: 非常に高い一致度
平均パターンとほぼ一致
60-80%: 良好な一致度
概ね平均パターンに沿っている
60%未満: 低い一致度
平均パターンから外れている可能性

位置-負荷相関分析

計算方法:
①平均位置の計算:
平均X座標 = Σ(X-POSF) / n
平均Y座標 = Σ(Y-POSF) / n
②位置誤差の計算（各データポイント）:
dx = X-POSF - 平均X座標
dy = Y-POSF - 平均Y座標
位置誤差 = √(dx² + dy²)
③合成負荷の計算（各データポイント）:
合成負荷 = √(X-TCMD² + Y-TCMD² + Z-TCMD²)
④相関係数の計算:
位置誤差と合成負荷のピアソン相関係数を算出
r = Σ((xi - x̄)(yi - ȳ)) / √(Σ(xi - x̄)² × Σ(yi - ȳ)²)
各指標の意味:
平均位置誤差:
全データポイントの位置誤差の平均値。加工軌跡の平均的なズレを示す。
最大位置誤差:
位置誤差の最大値。加工軌跡で最も大きくズレた箇所を示す。
平均負荷:
全データポイントの合成負荷の平均値。加工時の平均的な負荷レベル。
最大負荷:
合成負荷の最大値。加工中に最も負荷が高かった瞬間を示す。
変数の説明:
n: データポイント数
xi: 各データポイントの位置誤差
yi: 各データポイントの合成負荷
x̄: 位置誤差の平均値
ȳ: 合成負荷の平均値
Σ: 全データポイントの合計
相関係数の解釈:
|r| < 0.4: 弱い相関
位置と負荷の関係が弱い
0.4 ≤ |r| < 0.7: 中程度
|r| ≥ 0.7: 強い相関
位置ずれが負荷に大きく影響

位置別負荷ヒートマップ

計算方法:
①グリッド分割:
X-Y平面を20×20のグリッドに分割
各セルのサイズを算出
②合成負荷の計算（各データポイント）:
合成負荷 = √(X-TCMD² + Y-TCMD² + Z-TCMD²)
③セルごとの平均負荷を算出:
各セルに含まれるデータポイントの
合成負荷を平均化
④色分け表示:
青（低負荷）→緑→黄→赤（高負荷）
全ファイル統一のカラースケールを使用
各指標の意味:
セル内平均負荷:
そのグリッドセル内の全データポイントの合成負荷の平均値。値が高いほどその位置で高負荷が発生している。
グローバル最大負荷:
全ファイル・全セルの中で最も高い平均負荷の値。カラースケールの基準として使用される。
色の濃さ:
セル内平均負荷 ÷ グローバル最大負荷の比率で決定。0%（青）～100%（赤）
変数の説明:
X-POSF: X軸の実位置フィードバック値
Y-POSF: Y軸の実位置フィードバック値
X-TCMD: X軸のトルク指令値
Y-TCMD: Y軸のトルク指令値
Z-TCMD: Z軸のトルク指令値
ヒートマップの解釈:
赤色の領域: 負荷が非常に高い
異常発生の可能性が高い
黄色の領域: 中程度の負荷
注意が必要
青色の領域: 負荷が低い
安定した加工領域

セグメント別分析（時系列）

計算方法:
①初期オフセット適用:
オフセット位置を起点として
有効データ範囲を算出
②セグメント分割:
有効範囲を3つに分割：
・開始（0-33%）
・中盤（33-67%）
・終盤（67-100%）
③合成負荷の計算（各データポイント）:
合成負荷 = √(X-TCMD² + Y-TCMD² + Z-TCMD²)
④統計量の算出:
平均負荷 = Σ(L) / n
標準偏差 = σ
変動係数 = (σ / 平均負荷) × 100%
各指標の意味:
平均負荷:
セグメント内の合成負荷の平均値。負荷の大きさを表す。
最大負荷 / 最小負荷:
セグメント内での負荷の範囲。大きな差は不安定を示唆。
標準偏差（σ）:
負荷のバラツキの大きさ。大きいほど不安定。
変動係数（CV）:
標準偏差を平均で正規化した値。負荷の相対的な安定性を示す。
変数の説明:
X-TCMD: X軸のトルク指令値
Y-TCMD: Y軸のトルク指令値
Z-TCMD: Z軸のトルク指令値
オフセット: 加工開始位置の調整値
n: セグメント内のデータポイント数
σ: 標準偏差
変動係数（CV）の解釈:
CV < 30%: 安定した加工
正常な負荷パターン
30% ≤ CV < 50%: やや不安定
注意が必要
CV ≥ 50%: 非常に不安定
異常の可能性
